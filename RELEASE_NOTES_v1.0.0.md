# iBook v1.0.0 发布说明

## 首次正式发布

iBook 是一个专为网络小说阅读爱好者打造的本地化管理工具，帮助您记录、管理和追踪自己的阅读历程。

## 核心功能

### 小说信息管理
- **网络检索**：从 youshu.me 平台自动检索小说信息（书名、作者、字数、封面等）
- **手动录入**：支持完全自定义的手动录入方式
- **批量导入**：支持书名列表导入和文件上传导入，快速添加大量书籍

### 文档管理
- **多格式支持**：TXT、EPUB、PDF、DOC/DOCX、MOBI、AZW/AZW3
- **文档上传**：支持拖拽上传，自动保存到本地
- **字数统计**：自动统计文档字数，支持多种数据源（网络检索/文档统计/手动输入）

### 阅读状态管理
- **五种状态**：未读、阅读中、已读完、弃读、待读
- **快捷切换**：在书籍卡片上快速切换阅读状态
- **批量操作**：支持批量选择、批量删除、批量更新状态

### 标签系统
- **标签管理**：创建、编辑、删除标签
- **颜色标记**：8种默认颜色可选，方便分类识别
- **标签统计**：显示每个标签的使用数量
- **快速筛选**：通过标签快速筛选书籍

### 统计分析
- **总体统计**：总书籍数、总字数、已完成数、阅读中数等核心指标
- **数据可视化**：类型分布、平台分布、月度趋势、状态分布等图表
- **多维度分析**：按类型、平台、状态等维度统计

### 数据备份与恢复
- **完整备份**：备份数据库、文档、封面和设置
- **一键恢复**：从备份文件快速恢复所有数据
- **备份验证**：备份文件完整性检查

### 界面特性
- **明暗主题**：支持明亮、暗黑和跟随系统三种主题模式
- **响应式布局**：适配不同屏幕尺寸
- **网格/列表视图**：两种展示模式自由切换
- **搜索与筛选**：全文搜索、状态筛选、标签筛选

## 安装

### Windows
下载并运行 `iBook-1.0.0-setup.exe` 安装程序。

安装完成后，应用会自动创建桌面快捷方式。
