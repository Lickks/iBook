# 更新日志

本文档记录 iBook 项目的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2024-12-XX

### 新增

#### 核心功能
- 小说信息管理：支持网络检索、手动录入、批量导入
- 文档管理：支持 TXT、EPUB、PDF、DOC/DOCX、MOBI、AZW/AZW3 等多种格式
- 字数统计：自动统计文档字数，支持多种数据源（网络检索/文档统计/手动输入）
- 阅读状态管理：五种状态（未读、阅读中、已读完、弃读、待读）
- 标签系统：创建、编辑、删除标签，支持颜色标记和标签筛选
- 书架管理：创建自定义书架，支持书籍分类管理
- 统计分析：总体统计和数据可视化（饼图、折线图、柱状图）

#### 批量操作
- 批量导入：支持书名列表导入和文件上传导入
- 批量删除：支持批量选择并删除书籍
- 批量更新状态：支持批量修改阅读状态
- 批量添加标签：支持批量为书籍添加标签

#### 数据管理
- 数据备份：完整备份数据库、文档、封面和设置
- 数据恢复：从备份文件一键恢复所有数据
- 备份验证：备份文件完整性检查

#### UI/UX
- 明暗主题：支持明亮、暗黑和跟随系统三种主题模式
- 响应式布局：适配不同屏幕尺寸
- 网格/列表视图：两种展示模式自由切换
- 过渡动画：路由切换和列表操作动画
- 骨架屏：加载状态优化
- 虚拟滚动：长列表性能优化

#### 性能优化
- 数据库查询优化：批量查询、索引优化、分页查询、查询缓存
- 前端渲染优化：shallowRef、计算属性缓存、v-memo、虚拟滚动
- 防抖节流：搜索和筛选操作防抖优化
- 图片懒加载：优化图片加载性能

#### 搜索与筛选
- 全文搜索：支持书名、作者、描述搜索
- 多条件筛选：支持按状态、类型、平台、标签筛选
- 排序功能：支持按字数、时间、评分、书名、作者排序
- 升序/降序切换

### 技术实现

#### 后端
- Electron 主进程架构
- SQLite 数据库（better-sqlite3）
- IPC 通信机制
- 网络爬虫服务（youshu.me）
- 文件管理服务
- 字数统计服务（多格式支持）
- 封面下载服务
- 窗口状态管理

#### 前端
- Vue 3 Composition API
- Vue Router 路由管理
- Pinia 状态管理
- Element Plus UI 组件库
- ECharts 数据可视化
- TypeScript 类型支持

#### 构建工具
- Vite 构建工具
- electron-vite 集成
- electron-builder 打包
- TypeScript 编译
- ESLint 代码规范
- Prettier 代码格式化
