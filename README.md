# iBook - 个人网络小说阅读管理工具

一个基于 Electron + Vue 3 + TypeScript 开发的本地化小说管理工具，帮助网络小说阅读爱好者记录、管理和追踪自己的阅读历程。

## 核心功能

### 小说信息管理
- **网络检索**：从 youshu.me 平台自动检索小说信息（书名、作者、字数、封面等）
- **手动录入**：支持自定义录入所有书籍信息
- **批量导入**：支持书名列表导入和文件上传导入，快速添加大量书籍
- **信息编辑**：随时编辑和更新书籍信息

### 文档管理
- **多格式支持**：TXT、EPUB、PDF、DOC/DOCX、MOBI、AZW/AZW3
- **文档上传**：支持拖拽上传，自动保存到本地
- **字数统计**：自动统计文档字数，支持多种数据源（网络检索/文档统计/手动输入）
- **文档操作**：打开、删除、替换文档

### 阅读状态管理
- **五种状态**：未读、阅读中、已读完、弃读、待读
- **快捷切换**：在书籍卡片上快速切换阅读状态
- **批量操作**：支持批量选择、批量删除、批量更新状态
- **状态统计**：实时显示各状态的书籍数量

### 标签系统
- **标签管理**：创建、编辑、删除标签
- **颜色标记**：8种默认颜色可选，方便分类识别
- **标签统计**：显示每个标签的使用数量
- **快速筛选**：通过标签快速筛选书籍

### 统计分析
- **总体统计**：总书籍数、总字数、已完成数、阅读中数等核心指标
- **数据可视化**：类型分布、平台分布、月度趋势、状态分布等图表
- **多维度分析**：按类型、平台、状态等维度统计

### 数据备份与恢复
- **完整备份**：备份数据库、文档、封面和设置
- **一键恢复**：从备份文件快速恢复所有数据
- **备份管理**：查看备份信息，管理备份文件

### 界面特性
- **明暗主题**：支持明暗主题切换
- **响应式布局**：适配不同屏幕尺寸
- **网格/列表视图**：两种展示模式自由切换
- **搜索与筛选**：全文搜索、状态筛选、标签筛选

## 快速开始

### 系统要求
- Windows 10/11、macOS 10.13+ 或 Linux
- 至少 100MB 可用磁盘空间

### 安装方式

#### 方式一：下载安装包（推荐）
1. 访问 [Releases 页面](https://github.com/your-repo/ibook/releases)
2. 下载对应系统的安装包
3. 运行安装程序完成安装

#### 方式二：从源码构建
```bash
# 克隆仓库
git clone https://github.com/your-repo/ibook.git
cd ibook

# 安装依赖
npm install
# 或使用 pnpm（推荐）
pnpm install

# 启动开发模式
npm run dev

# 构建应用
npm run build

# 打包（Windows）
npm run build:win
# 打包（macOS）
npm run build:mac
# 打包（Linux）
npm run build:linux
```

## 使用手册

### 添加书籍

#### 方式一：网络检索添加
1. 点击首页的"添加书籍"按钮
2. 选择"网络检索"标签页
3. 输入书名（可选：同时输入作者）
4. 点击"搜索"按钮
5. 从搜索结果中选择目标书籍
6. 点击"确定导入"完成添加

#### 方式二：手动录入
1. 点击首页的"添加书籍"按钮
2. 选择"手动录入"标签页
3. 填写书籍信息（书名、作者、字数为必填项）
4. 上传封面图片（可选，支持拖拽）
5. 点击"保存"完成添加

#### 方式三：批量导入
1. 进入"批量导入"页面
2. 选择导入方式：
   - **书名列表导入**：在文本框中输入书名列表（每行一个）
   - **文件上传导入**：上传 TXT、EPUB、PDF 等格式的文档
3. 系统自动进行网络检索
4. 预览检索结果，选择要导入的书籍
5. 点击"批量导入"完成添加

### 管理书籍

#### 查看书籍详情
- 在首页点击书籍卡片或书名，进入详情页
- 查看完整的书籍信息、文档列表、标签等

#### 编辑书籍信息
1. 进入书籍详情页
2. 点击"编辑"按钮
3. 修改需要更新的信息
4. 点击"保存"完成更新

#### 删除书籍
- **单个删除**：在详情页点击"删除"按钮
- **批量删除**：在首页勾选多个书籍，点击"批量删除"

#### 更新阅读状态
- **快捷切换**：在书籍卡片上点击状态图标快速切换
- **详情页更新**：在详情页的状态下拉框中选择
- **批量更新**：在首页勾选多个书籍，点击"批量更新状态"

### 文档管理

#### 上传文档
1. 进入书籍详情页
2. 在"文档管理"区域点击"上传文档"
3. 选择文件或直接拖拽文件到上传区域
4. 系统自动保存文档并统计字数

#### 打开文档
- 在文档列表中点击"打开"按钮，使用系统默认应用打开文档

#### 删除文档
- 在文档列表中点击"删除"按钮，确认后删除文档

### 标签管理

#### 创建标签
1. 进入"标签管理"页面
2. 点击"新建标签"按钮
3. 输入标签名称，选择颜色
4. 点击"保存"完成创建

#### 为书籍添加标签
1. 进入书籍详情页或添加书籍页面
2. 在"标签"区域点击"添加标签"
3. 从列表中选择标签，或创建新标签
4. 保存后标签会显示在书籍卡片上

#### 筛选标签
- 在首页的标签列表中点击标签，快速筛选出包含该标签的书籍

### 统计分析

1. 进入"统计分析"页面
2. 查看总体统计数据和各类图表
3. 了解自己的阅读习惯和偏好

### 数据备份

#### 创建备份
1. 进入"设置"页面
2. 点击"数据备份"区域的"创建备份"按钮
3. 选择备份文件保存位置
4. 等待备份完成

#### 恢复备份
1. 进入"设置"页面
2. 点击"数据备份"区域的"恢复备份"按钮
3. 选择备份文件
4. 确认后等待恢复完成

## 常见问题

### 安装问题

**Q: 安装时提示"无法验证开发者"（macOS）**
A: 这是 macOS 的安全机制。请前往"系统偏好设置" > "安全性与隐私"，点击"仍要打开"。

**Q: 安装后无法启动应用**
A: 请检查系统版本是否满足要求，并尝试以管理员权限运行。

### 使用问题

**Q: 网络检索失败怎么办？**
A: 网络检索依赖 youshu.me 平台，如果检索失败，可以：
- 检查网络连接
- 尝试手动录入书籍信息
- 稍后重试

**Q: 文档上传失败**
A: 请检查：
- 文件格式是否支持（TXT、EPUB、PDF、DOC/DOCX、MOBI、AZW/AZW3）
- 文件大小是否超过限制（默认 100MB）
- 磁盘空间是否充足

**Q: 字数统计不准确**
A: 不同格式的文档统计方式不同：
- TXT：统计纯文本字符数
- EPUB/MOBI：解析电子书内容后统计
- PDF/DOCX：提取文本后统计
- 如果差异较大，可以手动输入字数

**Q: 如何批量导入大量书籍？**
A: 使用"批量导入"功能：
1. 准备书名列表（每行一个书名）
2. 或上传包含多个文档的文件夹
3. 系统会自动检索并导入

### 数据问题

**Q: 数据存储在哪里？**
A: 数据存储在应用数据目录：
- Windows: `%APPDATA%\ibook\`
- macOS: `~/Library/Application Support/ibook/`
- Linux: `~/.config/ibook/`

**Q: 如何备份数据？**
A: 在"设置"页面使用"数据备份"功能，会备份：
- 数据库文件
- 所有文档
- 封面图片
- 应用设置

**Q: 如何迁移到新电脑？**
A: 
1. 在原电脑创建备份
2. 将备份文件复制到新电脑
3. 在新电脑安装应用后恢复备份

**Q: 删除应用会丢失数据吗？**
A: 卸载应用不会自动删除数据目录，但建议在卸载前先备份数据。

### 性能问题

**Q: 应用启动较慢**
A: 首次启动需要初始化数据库，后续启动会更快。如果持续较慢，可以：
- 检查磁盘空间
- 清理不必要的文档
- 重启应用

**Q: 书籍列表加载慢**
A: 如果书籍数量很多（>1000本），可以：
- 使用搜索和筛选功能缩小范围
- 考虑清理不需要的书籍记录

### 其他问题

**Q: 支持哪些操作系统？**
A: 支持 Windows 10/11、macOS 10.13+ 和主流 Linux 发行版。

**Q: 是否支持云同步？**
A: 当前版本仅支持本地存储和备份恢复。云同步功能正在规划中。

**Q: 如何反馈问题或建议？**
A: 请在 GitHub 上提交 [Issue](https://github.com/your-repo/ibook/issues)。
